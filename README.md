# insurance-analytics-dashboard
An end-to-end Finance Analytics project using Power BI to develop an Insurance Premium and Payout KPI Dashboard. The project includes data transformation, KPI calculation, financial trend analysis, and interactive visualizations to support business decision-making in the insurance domain.
<br>
📌 Project Description

This project focuses on analyzing insurance company financial performance using real-world data. The task was to build a Power BI dashboard that showcases key performance indicators (KPIs) related to premium collection, claim payouts, and profitability trends.

This dashboard helps business users quickly evaluate how the insurance company is performing financially and identify patterns in revenue and payouts.
<br>
🎯 Project Goals

Analyze premium collections over time

Visualize payout and claim trends

Build KPIs that reflect business health

Enable interactive data exploration for stakeholders

🛠 Tools & Technologies
Technology	Use
Power BI	Data modeling, KPI calculations, visualization
Excel	Dataset storage and initial review
DAX	KPI logic and metrics
<br>
📂 Dataset Overview

The dataset includes financial details related to insurance operations such as:

Policy ID

Premium Amount

Claim Payout

Policy Type

Policy Start/End Dates

Customer Demographics

Note: This dataset is used for demonstration purposes and has been selected from a public or free source.

📊 Key Performance Indicators (KPIs)

The dashboard tracks the following:

Total Premium Collected

Total Claim Payouts

Net Revenue

Claims to Premium Ratio

Policy Count by Type

Monthly/Quarterly Trends

These indicators help monitor financial performance and risk exposure over time.

📈 Dashboard Features

Interactive Filters – Filter by policy type, date range, or geography

Summary KPI Cards – Quick snapshot of financial metrics

Trend Analysis – Visual charts to observe patterns

Comparison Views – Premium vs Payout insights

Segment Breakdown – Insights by customer or policy type

🔍 Key Business Insights

Premium collections can be compared month-over-month to detect growth or decline

Claim payouts should be analyzed against premium earnings to assess profitability

Policy types or customer segments may reveal higher risk exposure

These insights help insurance decision-makers plan strategies, control risk, and improve financial outcomes.
